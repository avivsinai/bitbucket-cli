# Privacy Policy — bkt (Bitbucket CLI)

**Effective date**: 2026-04-04

## Data collection

bkt does not collect, transmit, or store any personal data, telemetry, or
usage analytics. All API requests go directly from the user's machine (or CI
runner) to the configured Bitbucket instance.

## Credentials

Authentication tokens are stored locally in the user's OS keychain or an
encrypted file. They are never transmitted to any party other than the
configured Bitbucket host. bkt does not proxy, relay, or cache tokens on any
third-party server.

## Third-party services

bkt does not integrate with any third-party analytics, advertising, or tracking
services.

## Changes

Updates to this policy will be posted in the repository
[CHANGELOG](https://github.com/avivsinai/bitbucket-cli/blob/master/CHANGELOG.md).
